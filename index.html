Variables

we create variables whenever we want to store the value of something so that we can re-use that value later by simply referencing the variable name

declaration & assignment

								
									var message = "Hello class";

until otherwise assigned `message` will have the value of "Hello class"

a variable can be assigned to one of the JavaScript data types...
